# Use Case Definer Agent

## Overview
The Use Case Definer Agent transforms high-level workload descriptions into detailed, actionable use cases with specific questions your Neptune solution can answer.

## Input
- Topic sentence or short statement about your workload
- Brief description of the domain or problem space

## Output
- Detailed use case specification
- List of specific questions the solution should answer
- Context and requirements for the graph model

## Usage
Provide a concise description of your workload, and the agent will expand it into a comprehensive use case with clear objectives and measurable outcomes.

## Example
**Input**: "Financial fraud detection system"
**Output**: Detailed use case covering transaction monitoring, suspicious pattern identification, and compliance reporting requirements.

## Cost
This agent generally costs under $0.01 USD.
