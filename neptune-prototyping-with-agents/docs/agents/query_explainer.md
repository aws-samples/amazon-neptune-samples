# Query Explainer Agent

## Overview
The Query Explainer Agent retrieves and analyzes query explain plans, providing detailed breakdowns of query execution and improvement recommendations.

## Input
- openCypher query
- Neptune cluster connection via MCP Server
- Performance analysis requirements

## Output
- Step-by-step query execution breakdown
- Performance metrics and analysis
- Optimization recommendations
- Bottleneck identification

## Usage
Submit any openCypher query to get a comprehensive analysis of how Neptune executes it, including performance characteristics and suggestions for improvement.

## Features
- Explain plan retrieval and analysis
- Performance bottleneck identification
- Step-by-step execution breakdown
- Actionable optimization recommendations
- Integration with Neptune MCP Server
