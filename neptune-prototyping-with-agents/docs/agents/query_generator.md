# Query Generation Agent

## Overview
The Query Generation Agent converts natural language questions into valid graph queries for your Neptune database.

## Input
- Natural language question
- Graph model context

## Output
- Valid openCypher query
- Query validation via Neptune MCP Server
- Explanation of query logic

## Usage
Ask questions in natural language about your data, and the agent will generate the appropriate openCypher query. All queries are validated against your Neptune cluster to verify correctness.

## Features
- Natural language to openCypher conversion
- Real-time query validation
- Query explanation and documentation

## Cost
This agent generally costs under $1 USD per execution.
